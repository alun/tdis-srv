h2. Сервисы ТДИС

Целью данного проета является содействие в изучении "ТДИС":http://newasp.omskreg.ru/tdis/.

В настоящий момент продукт реализует следующие сервисы:
* поиск и расшифровка аббревиатур
* текстовый фильтр реализующий замену аббревиатур в тексте на интерактивные элементы, позволяющие узнавать расшифровки при наведении курсора мыши
